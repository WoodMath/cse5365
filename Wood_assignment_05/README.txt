File 'pyramid_05.txt' will be loded by default.

A different file can be loaded 1 of 2 ways.

1) By commandline arguement i.e.
	$ python Wood_Assignment_05.py teapot_05.txt
	
2) By pressing the 'n' key, whcih will result in a input prompt in the Python Console.
	Note: After the user presses 'n', they will have to shift focus to the Python Console, type the name of the file (followed by enter), and then shift focus back to the OpenGL window.
